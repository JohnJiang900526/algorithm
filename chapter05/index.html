# 链表

什么是链表

链表存储有序的元素集合，但是不同于数组，链表中的元素在内存中并不是连续放置的。

每一个元素由存储元素本身的节点和一个指向下一个元素的引用(也称为指针或者链接)组成。

相对于传统的数组，链表的好处在于，添加和删除元素的时候不需要移动其他元素。然而

链表需要指针，因此实现链表时需要额外的注意。数组的另一个细节是可以直接访问任何

位置的元素，而要访问链表中的一个元素，需要起点(表头)开始迭代直到找到所需元素。
