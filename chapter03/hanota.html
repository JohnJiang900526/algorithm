# 汉诺塔

汉诺塔

汉诺塔（又称河内塔）问题是源于印度一个古老传说的益智玩具。 塔上有三根柱子和一套直径各不相同的空心圆盘，开始时源柱子上的所有圆盘都按从大到小的顺序排列。 目标是通过每一次移动一个圆盘到另一根柱子上，最终把一堆圆盘移动到目标柱子上，过程中不允许把 较大的圆盘放置在较小的圆盘上；

寻找规律（把所有的圆盘移动到C）：

1）n(圆盘个数) == 1

第一次：1号盘 A -> C sum(移动次数) = 1

2）n == 2

第一次：1号盘 A -> B

第二次：2号盘 A -> C

第三次：1号盘 B -> C　　sum = 3

3）n == 3

第一次：1号盘 A -> C

第二次：2号盘 A -> B

第三次：1号盘 C -> B

第四次：3号盘 A -> C

第五次：1号盘 B -> A

第六次：2号盘 B -> C

第七次：1号盘 A -> C　　sum = 7

以此类推...

故不难发现规律,移动次数为：sum = 2^n - 1

算法递归

把一堆圆盘从一个柱子移动另一根柱子，必要时使用辅助的柱子。可以把它分为三个子问题：

首先，移动一对圆盘中较小的圆盘到辅助柱子上，从而露出下面较大的圆盘，

其次，移动下面的圆盘到目标柱子上

最后，将刚才较小的圆盘从辅助柱子上在移动到目标柱子上

把三个步骤转化为简单数学问题：

（1） 把 n-1个盘子由A 移到 B；

（2） 把 第 n个盘子由 A移到 C；

（3） 把n-1个盘子由B 移到 C；

例如:有A上有4个盘子

(1) 把 1-3由 A 移动到 B

(2) 把 4 由 A 移动到 C

(3) 把 1-3 由 B 移动到 C

代码实现

// 栈
function Stack() {
  let items = [];

  // 实现可以添加一个或者多个新元素到栈顶
  this.push = function () {
    let args = arguments;

    for (let i = 0; i < args.length; i++) {
      items[items.length] = args[i];
    }

    return args[args.length];
  };

  // 实现pop方法，移除栈顶元素。同时返回被移除的元素。
  this.pop = function () {
    return items.pop();
  };

  // 实现peek方法，返回栈顶元素，不会移除栈顶元素。
  this.peek = function () {
    return items[items.length - 1];
  };

  // 实现isEmpty方法，如果没有任何元素返回true，否则false
  this.isEmpty = function () {
    return items.length === 0;
  };

  // 实现clear方法，移除栈里的所有元素。
  this.clear = function () {
    items = [];
  };

  // 实现size方法，和数组的length属性类似。
  this.size = function () {
    return items.length;
  };

  // 实现print方法，输出整个队列信息。
  this.print = function () {
    return items.toString();
  };
}

// 汉诺塔逻辑实现
let stackMap = {
  "A": new Stack(),
  "B": new Stack(),
  "C": new Stack(),
};


for (let i = 3; i >= 1; i--) {
  stackMap["A"].push(i);
}

function hanoi(disc, a, b, c) {
  if (disc === 0) {
    return false;
  }

  hanoi(disc - 1, a, c, b);
  console.log(' 移动 '+ disc +  ' 号圆盘 ' + ' 从 ' + a +  ' 移动到 ' +  c);
  stackMap[c].push(stackMap[a].pop());
  hanoi(disc - 1, b, a, c);
}

console.log(stackMap["A"].print());
console.log(stackMap["B"].print());
console.log(stackMap["C"].print());


hanoi(stackMap["A"].size(), "A", "B", "C");

console.log(stackMap["A"].print());
console.log(stackMap["B"].print());
console.log(stackMap["C"].print());
