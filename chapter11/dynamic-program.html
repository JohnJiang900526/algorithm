# 动态规划

动态规划

动态规划(DP)是一种将复杂问题分解成更小的子问题来解决的优化技术。

动态规划著名的问题：

  1. 背包问题
给出一组项目，各自有值和容量，目标找出总值最大的项目的集合。
这个问题限制是总容量必须小于等于背包容量。
  1. 最长公共子序列问题
找出一组序列的最长公共子序列，可由另一序列删除元素但是不改变余下元素的顺序得到。
  1. 矩阵链相乘
给出一系列矩阵，目标是找到这些矩阵相乘的最高效的办法(计算次数尽可能的少)。
相乘操作不会进行，解决方案是找到这些矩阵的各自相乘的顺序。
  1. 硬币找零
给出面额为d1, d2, d3, ... dn的一定数量的硬币和要找零的钱数，找出多少种找零的方法。
  1. 图的全源最短路径
对所有顶点对(u, v),找出从顶点u到顶点v的最短路径。

// 最少找零的变种问题
function MinCoinChange(arr) {
  let coins = arr;
  let cache = {};

  this.makeChange = function (amount) {
    let that = this;

    if (!amount) {
      return [];
    }

    if (cache[amount]) {
      return cache[amount];
    }

    let min = [];
    let newMin, newAmount;
    for (let i = 0; i < coins.length; i++) {
      let coin = coins[i];
      newAmount = amount - coin;

      if (newAmount >= 0) {
        newMin = that.makeChange(newAmount);
      }

      if (
        newAmount >= 0 &&
        (newMin.length < min.length - 1 || !min.length) &&
        (newMin.length || !newAmount)
      ) {
        min = [coin].concat(newMin);
      }
    }

    return (cache[amount] = min);
  }
}

let minCoinChange = new MinCoinChange([1, 3, 4]);

console.log(
  minCoinChange.makeChange(6)
); // [ 3, 3 ]
