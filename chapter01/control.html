# 控制结构

JavaScript的控制结构与C和java中的类似。条件语句支持if ... else 和 switch。 循环支持while、do...while、for。

# 条件语句

1. if、if else、 if else if else

// if
let num = 1;
if (num === 1) {
  console.log(true);
}

// if else 
let num = 1;
if (num === 1) {
  console.log(true);
} else {
  console.log(false);
}

// if else if else 
let num = 1;
if (num === 1) {
  console.log(true);
} else if (num === 2) {
  console.log(true);
} else {
  console.log(false);
}

2. switch语句

let key = 0;

switch (key) {
  case 0:
    console.log(0);
    break;
  case 1:
    console.log(1);
    break;
  case 2:
    console.log(2);
    break;
  default:
    console.log(0);
}

注意

对于switch语句来说，case和break关键字是非常重要的。case判断switch的值和case分支语句是否相等。 break是终止switch语句执行。没有break，case语句会继续执行，一直到有break关键字，如果没有会执行 到底。

# 循环

for循环

for(let i = 0; i < 10; i++) {
  console.log(i);
}

do ... while 循环

let i = 0;
do {
  console.log(i);
  i++;
} while (i < 10);

while 循环

let i = 0;
while (i < 10) {
  console.log(i);
  i++;
}
