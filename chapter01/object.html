# 面向对象编程

JavaScript里对象是普通键值对的集合。 创建一个普通对象有两种形式。

第1种

var obj = new Object();

第2种

var obj = {};

obj = {
  name: "john",
  work: {
    isLeader: true,
    position: "总经理"
  },
  age: 30
};

在面向对象编程(OOP)中，对象是一个类的实例。 一个类定义了对象的特征。我们可以在这个类的里面 来表达各种算法和数据结构。

比如：

function Book(title, pages, isbn) {
  this.title = title;
  this.pages = pages;
  this.isbn = isbn;
}

实例化这个类：

let book = new Book("title", 200, "isbn");

console.log(book.title);
console.log(book.pages);
console.log(book.isbn);

类可以包含函数：

Book.prototype.getTitle = function () {
  console.log(this.title);
}

book.getTitle();

也可以写在声明函数里：

function Book(title, pages, isbn) {
  this.title = title;
  this.pages = pages;
  this.isbn = isbn;
  this.getTitle = function () {
    console.log(this.title);
  }
}

let book = new Book("title", 200, "isbn");
book.getTitle();
